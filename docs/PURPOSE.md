# Purpose

## Problem being solved

<!-- TODO: fill in — 1–3 paragraphs describing the problem this project solves -->

Managing personal finances requires importing transaction data from multiple banks and credit cards, each producing differently-formatted PDFs. Existing tools either require manual data entry, charge subscription fees, or send data to third-party servers. This project provides a local-first desktop tool that accepts raw bank statements and produces categorized, queryable transaction data — entirely on the user's machine.

## Non-goals

<!-- TODO: fill in — list things this project explicitly does not try to do -->

- Not a budgeting tool (no envelopes, goals, or spend limits)
- Not a multi-user or cloud-synced application
- Not a replacement for accounting software (no double-entry bookkeeping)
- Not a tax preparation tool

## Intended audience

<!-- TODO: fill in — who is this for? -->

Personal use by a single technically-proficient user who wants full control over their financial data without relying on third-party services.
